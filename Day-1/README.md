# ☀️ Summer PEP — Day 1

Welcome to **Day 1** of the Summer Professional Experience Program (PEP)!

This repository contains the introductory project(s) created during the first day of the program. The focus is to get familiar with the Spring Boot ecosystem and setup basic web application architecture.

---

## 📁 Projects Included

### 1. SpringBootDemo
A foundational **Spring Boot 4.1** application demonstrating the setup of a basic web project.

**Tech Stack**:
- Java 17
- Spring Boot (WebMVC)
- Spring Boot DevTools (for hot reloading)

**Key Learnings**:
- Initializing a Spring Boot application using Maven.
- Understanding the `pom.xml` dependencies (like `spring-boot-starter-webmvc`).
- Running a basic Spring application.

---

## 🚀 How to Run

1. Navigate to the `SpringBootDemo` folder.
2. Ensure you have Java 17 and Maven installed.
3. Run the application:
   ```bash
   ./mvnw spring-boot:run
   ```
4. Access the server (default port `8080` unless configured otherwise) via browser or Postman.

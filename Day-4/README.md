# ☀️ Summer PEP — Day 4

Welcome to **Day 4** of the Summer Professional Experience Program (PEP)!

Day 4 marks an important transition from single-node/monolithic applications to learning about basic structure organization and preparing for Microservices.

---

## 📁 Projects Included

### 1. summerpepday-4-main
This folder contains the core exercises and applications developed during Day 4. 
The focus was on cleaning up project structures, understanding service-oriented approaches, and ensuring that our code is modular enough to be broken down into microservices in the upcoming days.

**Key Learnings**:
- Modularizing code structure.
- Refactoring existing Monoliths to extract domain logic.
- Building robust RESTful endpoints with appropriate status codes and DTOs (Data Transfer Objects).

---

## 🚀 How to Run

1. Navigate to the specific inner project directory.
2. Run the application using the Maven wrapper:
   ```bash
   ./mvnw spring-boot:run
   ```
3. Test the exposed REST APIs using Postman or your browser.

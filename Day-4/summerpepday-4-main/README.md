- XML Configuration

## Project

### Test

A basic Spring Framework application demonstrating how to create and manage Spring Beans using XML configuration.

### Features

- Bean configuration using `applicationContext.xml`
- Loading the Spring container using `ApplicationContext`
- Creating and accessing `Employee` objects as Spring Beans
- Understanding Dependency Injection and IoC principles

# ☀️ Summer PEP — Day 7

Welcome to **Day 7** of the Summer Professional Experience Program (PEP)!

Today's focus is on streamlining Java development by reducing boilerplate code using **Lombok**.

---

## 📁 Projects Included

### 1. LombokDemo
A standalone project showcasing the power of Project Lombok in a Spring Boot application.
- Uses annotations like `@Data`, `@Getter`, `@Setter`, and `@NoArgsConstructor` to keep Entities and DTOs clean and concise.
- Eliminates the need to manually write standard Getters/Setters or toString() methods!

---

## 🚀 How to Run

1. Navigate to the `LombokDemo` folder.
2. In your IDE, ensure that **Annotation Processing** is enabled, and the Lombok plugin is installed.
3. Run the application using the Maven wrapper:
   ```bash
   ./mvnw spring-boot:run
   ```
